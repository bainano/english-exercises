Python安装指南

注意：检测到您的系统中未安装Python或版本过低，无法运行HTTP服务器。请按照以下步骤安装Python。

Windows系统安装步骤

  1. 访问Python官方网站：https://www.python.org/downloads/windows/
  2. 在页面中找到"Stable Releases"部分
  3. 点击最新的Python版本下的"Download Windows installer (64-bit)"链接
  4. 下载完成后，双击安装包开始安装
  5. 在安装界面中，请务必勾选"Add Python to PATH"选项
  6. 选择"Install Now"进行安装
  7. 安装完成后，按Win+R键，输入"cmd"打开命令提示符
  8. 输入"python --version"验证安装是否成功
重要提示：如果在安装过程中没有勾选"Add Python to PATH"，您需要手动添加或重新安装。

macOS系统安装步骤

  1. 访问Python官方网站：https://www.python.org/downloads/macos/
  2. 点击最新的Python版本下的macOS安装包下载链接
  3. 下载完成后，双击.pkg文件开始安装
  4. 按照安装向导完成安装过程
  5. 打开终端(Terminal)，输入"python3 --version"验证安装是否成功

Linux系统安装步骤

大多数Linux发行版都可以通过包管理器安装Python：

Ubuntu/Debian系统： 
sudo apt update
sudo apt install python3 python3-pip
Fedora系统： 
sudo dnf install python3 python3-pip
CentOS/RHEL系统： 
sudo yum install python3 python3-pip

验证安装

安装完成后，请打开命令行工具（Windows: 命令提示符 或 PowerShell，macOS/Linux: 终端）并输入以下命令验证：

python --version

或

python3 --version

如果显示类似"Python 3.x.x"的版本信息，说明安装成功。

常见问题：
  • 如果提示"python不是内部或外部命令"，请检查是否勾选了"Add Python to PATH"
  • 某些系统可能需要重启后才能正常使用python命令
  • Linux系统通常使用"python3"命令而不是"python"

安装完成后，请重新运行启动脚本来启动HTTP服务器。
