# 英语学习应用

## 应用概述

这是一款交互式英语学习应用，专为提高用户的英语阅读和翻译能力设计。应用提供了丰富的练习题，用户可以通过拖拽单词来完成句子翻译，从而提升英语水平。

## 功能特性

- **交互式练习**：通过拖拽单词块完成句子翻译
- **题库管理**：支持查看和管理练习题库
- **进度跟踪**：显示当前练习进度和完成情况
- **即时反馈**：提供答案检查和提示功能
- **响应式设计**：适配不同屏幕尺寸的设备
- **一键启动**：提供跨平台的启动脚本，使用简单便捷

## 快速开始

### 使用一键启动脚本

#### Windows系统
1. 确保已安装Python 3.x版本
2. 双击运行 `start_app.bat` 文件
3. 或使用桌面快捷方式 `En-start-win.lnk`

#### Linux/Mac系统
1. 确保已安装Python 3.x版本
2. 赋予脚本执行权限：`chmod +x start_app.sh`
3. 运行脚本：`./start_app.sh`
4. 或使用快捷方式 `En-satrt-mac.lnk`

### 脚本功能

- **一键启动**：自动启动HTTP服务器并在默认浏览器中打开应用
- **环境检查**：验证Python安装和版本
- **文件验证**：检查必要的应用文件是否存在
- **端口检测**：检查端口8000是否被占用
- **错误处理**：提供详细的错误信息和解决方案
- **用户反馈**：清晰的状态提示和进度显示
- **日志记录**：记录启动过程和运行状态

## 使用指南

### 答题界面
1. 在主页面，您将看到一段需要翻译的中文句子
2. 下方有多个英语单词块
3. 拖拽单词块到上方的答题区域，组成正确的英文句子
4. 点击"检查"按钮验证答案
5. 点击"重新开始"按钮重置当前题目

### 题库管理
1. 点击顶部的"题库"按钮进入题库页面
2. 查看所有可用的练习题
3. 选择特定题目进行练习

## 日志说明

- **启动日志**：保存在 `logs` 目录中，文件名格式为 `start_app_日期_时间.log`
- **服务器日志**：保存在 `logs/server.log`
- 日志包含启动过程、环境信息和可能的错误记录，便于问题排查

## 注意事项

- 运行脚本时会显示一个命令窗口，请不要关闭它，关闭窗口将停止HTTP服务器
- 如需手动停止服务，请按任意键关闭启动窗口
- 确保8000端口未被其他程序占用
- 如需更改端口号，请修改启动脚本中的端口配置

## 故障排除

1. **找不到Python环境**：请先安装Python 3.x版本
2. **找不到应用文件**：确保在正确的应用目录下运行脚本
3. **端口被占用**：关闭占用8000端口的程序，或修改脚本中的端口配置
4. **浏览器未自动打开**：请手动访问 http://localhost:8000/index.html
5. **答题区域无反应**：请尝试刷新页面或清除浏览器缓存
6. **题库加载失败**：检查test_bank.json文件是否存在且格式正确

## 系统要求

- 操作系统：Windows、macOS或Linux
- Python 3.x（用于运行HTTP服务器）
- 支持HTML5和JavaScript的现代浏览器（Chrome、Firefox、Safari、Edge等）
- 至少512MB内存
- 足够的磁盘空间存储应用文件和日志
